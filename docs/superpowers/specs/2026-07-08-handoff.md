# 2026-07-08 Session Handoff — 四模块股票分析系统设计总结

## 目标
为 hermesStockAgent 项目重新设计四模块股票分析系统，产出完整规格文档供下一个 session 直接实施。

## 用户需求（逐轮确认收集）
- **场景**: 个人决策 + 主动选股 + 深度研究 + 市场感知（全部4个）
- **市场**: A股 + ETF/基金
- **维度**: 基本面、技术信号、行业概念、行情表现（全部）
- **周期**: 短线 + 中长线
- **操作建议**: 多维度分析 + 含价格区间的总结
- **消息面**: 新闻情感 + 龙虎榜 + 资金流向 + 板块轮动 + 公告AI解读 + 大单资金分析
- **暗盘**: 降级为"大单资金流分析"（无Level-2数据）

## 选股器OCP架构设计
```python
# 策略注册表
class ScreenerStrategyRegistry:
    def register(self, name: str, fields: list[FilterField])
    def get_strategy(self, name: str) -> ScreenerStrategy
    def list_strategies() -> list[str]

# 声明式筛选字段
class FilterField(BaseModel):
    name: str          # "roe", "pe_ttm", "macd_golden_cross"
    label: str         # "ROE", "市盈率", "MACD金叉"
    type: str          # "number_range", "boolean", "choice"
    category: str      # "基本面", "技术信号", "行情表现", "行业概念"
    constraints: dict  # {"min": 0, "max": 100, "step": 1}

# 后端API
GET /api/v1/screener/fields       → list[FilterField]  (前端动态渲染)
POST /api/v1/screener/run         → list[StockResult]
GET /api/v1/screener/skills       → list[SkillTemplate]
```
