## 模块1: 选股器

**数据依赖:** 需要 PE/PB/ROE/行业/市值/股息率。当前同花顺 API 的 PE/PB 标注"敬请期待"，ROE 和负债率已有。

**替代数据源方案:**
- AKShare（免费，覆盖全）但之前网络不可用
- 东方财富/雪球 — 需要爬虫
- 暂用已有数据（ROE + 负债率 + 行业 + 价格）做简化版筛选

**UI:** 独立页面，顶栏筛选条件 → 表格结果 → 点击进入个股页面

## 模块2: 深度诊断

**数据依赖:** 
- 多周期K线 — 同花顺 API 当前仅支持日线，周线/月线需自算（从日线聚合）
- 股票对比 — 需要同时展示两支股票的K线+指标

**UI:** 复用当前图表区域，顶部加对比选择和周期切换

## 模块3: 操作建议

**数据依赖:**
- 买卖信号: MACD/KDJ/RSI 已有
- 支撑阻力: 布林带已有，斐波那契可自算
- ATR 止损: 需自算（从日线计算 ATR(14)）
- 仓位: 纯计算，不依赖外部数据

**UI:** 右侧面板信号列表 + 关键价位标注在K线图上

## 模块4: 消息情报

**数据依赖:**
- 龙虎榜: 同花顺 API 有 `dragon-tiger-list` 接口
- 北向资金: 需额外数据源（AKShare 有但不可靠）
- 板块轮动: 需行业分类 + 涨跌幅排名
- 公告: 同花顺 API 有"个股异动原因"接口

**UI:** 独立信息面板，可切换龙虎榜/资金流向/板块轮动

## 实施优先级

| 优先级 | 模块 | 理由 |
|--------|------|------|
| P0 | 模块3 操作建议 | 已有数据，可立即实现，用户最直接受益 |
| P1 | 模块2 深度诊断 | 周线聚合 + 对比模式，相对独立 |
| P2 | 模块1 选股器 | 需要解决 PE/PB 数据源问题 |
| P3 | 模块4 消息情报 | 依赖外部 API 较多，最复杂 |

## 技术约束

- 所有数据只能通过同花顺 API 获取
- PE/PB 暂不可用 → P2 选股器先做简化版
- 多周期K线从日线数据聚合自算
- 不用 AKShare（网络不稳定）
